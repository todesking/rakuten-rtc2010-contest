■実行方法
●Eclipseを利用したコンパイルと実行
1) 上部メニューのFile -> Import -> General -> Existing Projects into Workspaceの順に選択
2) Browseボタンを押して、解凍したサンプルプロジェクトのフォルダを選択
3) Finishボタンをクリックしてサンプルプロジェクトをワークスペースにインポート
4) インポートしたサンプルプロジェクト -> Default package -> Starter.java上で右クリック
5) ポップアップメニュー内でRun as -> Java Applicationをクリックして実行

●コマンドラインからのコンパイルと実行
1) compile_run.batをダブルクリック
※必要であれば中身を確認して編集してください。


■開発方法
1) あなたのチーム名.java をチーム名にリネーム
2) Starter.java で3.1でリネームしたクラスのインスタンスをGameStarter.startに渡す
3) リネームしたファイルを編集してAIを開発
